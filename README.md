# mvn-repo
Personal Maven Repository

# Usage
pom.xml
```xml
<repositories>
    <repository>
        <id>snapshots-repo</id>
        <url>https://raw.githubusercontent.com/dengliming/mvn-repo/master/snapshots</url>
    </repository>
    <repository>
        <id>releases-repo</id>
        <url>https://raw.githubusercontent.com/dengliming/mvn-repo/master/releases</url>
    </repository>
</repositories>
```
